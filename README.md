# Power 11 - Arcade Puzzle Battle

## Installation
Just download **Power11.exe**, place it anywhere, and double-click to play. No installation required.

## Controls
- **Arrows / WASD**: Move
- **Enter**: Confirm
- **F11**: Fullscreen
- **Esc**: Menu

## Features
- **Versus AI**: Challenge the high-performance Expectimax AI.
- **Leaderboards**: Compete for the highest score worldwide.
- **Time Attack**: Race against the clock.
